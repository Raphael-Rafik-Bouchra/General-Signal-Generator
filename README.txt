1.Program explanation:
-This program is general signal generator that allows the user to do some operations on the output signal.
-The user enters the sample rating (fs) and starting time (ti) and finishing time (tf) and number of breakpoints (n) and their positions.
-Then the program asks the user to enter the specifications of the signal of the current region.
-Then the program plots the total signal for the user.
-Then the program asks the user to do some operations on the signal (The operations are cumulative).
2.validations:
- The program validates all data input values
- The program gives a descriptive example to the user to follow
- The program DOES NOT validate null data input
- The code is full of comments that explain each part of the code
